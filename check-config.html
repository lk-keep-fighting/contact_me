⚠️ 问题诊断：验证邮件跳转错误

问题：注册验证邮件链接跳转到 localhost:3000

应该跳转到：https://contact-me.dev/

原因：Supabase项目的Site URL配置错误

🚨 需要立即修复的配置

1. 在Supabase Dashboard中修复Site URL

  1. 登录 Supabase Dashboard
  2. 选择你的项目
  3. 导航到 Settings → Authentication
  4. 在 "Site URL" 字段中，将：
    5. ❌ 当前值: http://localhost:3000
    ✅ 改为: https://contact-me.dev

2. 配置允许的重定向URL

在同一页面的 "Additional Redirect URLs" 部分添加：

https://contact-me.dev/**
https://contact-me.dev/dashboard.html
https://contact-me.dev/login.html
https://contact-me.dev/redirect.html

🧪 配置验证测试

✅ 修复完成后的验证步骤

  1. 在Supabase中保存新的配置
  2. 等待1-2分钟让配置生效
  3. 注册一个新的测试账号
  4. 检查验证邮件中的链接是否指向正确的域名
  5. 点击验证链接，确认能正确跳转到你的网站
